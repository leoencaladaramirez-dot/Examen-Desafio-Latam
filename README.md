# Resumen del Proyecto

Este proyecto es un portafolio personal de Marlon Encalada, desarrollado utilizando HTML, CSS, y Bootstrap. El objetivo principal es mostrar la trayectoria profesional, educación, experiencia laboral y trabajos destacados en el campo del periodismo y la comunicación.

# Logros

1. **Estructura HTML Semántica**: Se creó una estructura clara y organizada utilizando etiquetas HTML5 apropiadas para cada sección del portafolio.
   
2. **Diseño Responsivo**: Gracias a Bootstrap, el diseño se adapta a diferentes tamaños de pantalla, mejorando la experiencia de usuario en dispositivos móviles y de escritorio.

3. **Estilo Personalizado**: Se implementaron estilos únicos mediante CSS, incluyendo una paleta de colores armoniosa y fuentes seleccionadas para una mejor legibilidad.

4. **Secciones Clave**:
   - **Hero**: Presentación destacada con una imagen y mensaje de bienvenida.
   - **Sobre mí**: Resumen personal que coincide con la misión profesional.
   - **Educación**: Resumen de formación académica y certificaciones obtenidas.
   - **Experiencia**: Trayectoria profesional organizada cronológicamente.
   - **Portafolio**: Exhibición de trabajos destacados con imágenes y descripciones.
   - **Contacto**: Información de contacto y enlaces a redes sociales.

5. **Optimización de Código**: Se limpió el CSS para evitar conflictos y se unificaron estilos, asegurando una visualización correcta de los textos.

# Siguientes Pasos

- *Publicación en GitHub Pages*: El proyecto se puede desplegar facilmente mediante GitHub Pages para que esté disponible en línea.
- *Mejoras Futuras*: Posibilidad de agregar más proyectos al portafolio y secciones adicionales según la evolución de la carrera profesional.

Experticia en comunicación y un portafolio bien presentado serán la clave para destacar en el ámbito laboral.
